# Divine Aura - Product Requirements Document

## Original Problem Statement
Build a premium, modern, luxurious website for a clean beauty brand named "Divine Aura". MVP homepage that establishes trust + differentiation BEFORE any Aura quiz. Products are NOT finalized yet.

## Target Audience
Modern, conscious beauty consumers interested in clean, luxury beauty products who value transparency, intentionality, and conscious beauty practices.

## User Personas
1. **Conscious Consumer**: Values ingredient transparency, clean formulations, small-batch quality
2. **Luxury Seeker**: Appreciates premium aesthetics, elegant design, sophisticated branding
3. **Early Adopter**: Wants early access to launches, interested in aura-led personalization

## Core Requirements (Static)
- Single-page homepage with key sections
- About page with brand story
- Contact page with form
- No products page (products not finalized)
- SEO basics (title, meta, OpenGraph)
- Mobile responsive design
- Waitlist capture (inline + modal)
